# 三角洲租号平台 - 香港服务器部署指南

## 一、准备工作

### 1. 服务器环境要求
- **操作系统**: Ubuntu 20.04+ / CentOS 7+
- **域名**: 已解析到香港服务器IP
- **端口**: 80 (HTTP), 443 (HTTPS), 22 (SSH)

### 2. 本地准备
- 下载 `delta-rent-v2.zip` 文件
- 准备 SSH 登录信息（IP、用户名、密码/密钥）

---

## 二、服务器初始化

### 1. 连接服务器
```bash
# 使用密码登录
ssh root@你的服务器IP

# 或使用密钥登录
ssh -i /path/to/your/key.pem root@你的服务器IP
```

### 2. 更新系统
```bash
# Ubuntu/Debian
apt update && apt upgrade -y

# CentOS
yum update -y
```

### 3. 安装必要软件
```bash
# 安装 Nginx
apt install -y nginx

# 安装 unzip（用于解压）
apt install -y unzip

# 启动 Nginx
systemctl start nginx
systemctl enable nginx
```

---

## 三、上传网站文件

### 方法一：使用 SCP（推荐）

**在本地终端执行：**
```bash
# 上传压缩包到服务器
scp delta-rent-v2.zip root@你的服务器IP:/root/

# 登录服务器
ssh root@你的服务器IP

# 解压到网站目录
cd /root
unzip delta-rent-v2.zip -d /var/www/
mv /var/www/delta-rent/dist/* /var/www/html/

# 设置权限
chown -R www-data:www-data /var/www/html
chmod -R 755 /var/www/html
```

### 方法二：使用 FTP/SFTP
使用 FileZilla 等工具连接服务器，将 `dist` 文件夹内容上传到 `/var/www/html/`

## 四、配置 Nginx

### 1. 创建配置文件
```bash
nano /etc/nginx/sites-available/delta-rent
```

### 2. 粘贴以下配置
```nginx
server {
    listen 80;
    server_name 你的域名.com;  # 修改为你的域名
    
    root /var/www/html;
    index index.html;
    
    # Gzip 压缩
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types text/plain text/css text/xml text/javascript application/javascript application/xml+rss application/json;
    
    # 缓存静态资源
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    # 前端路由支持（重要！）
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 安全响应头
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;
}
```

### 3. 启用配置
```bash
# 创建软链接
ln -s /etc/nginx/sites-available/delta-rent /etc/nginx/sites-enabled/

# 删除默认配置
rm -f /etc/nginx/sites-enabled/default

# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx
```

## 五、配置 HTTPS（SSL证书）

### 1. 安装 Certbot
```bash
# Ubuntu
apt install -y certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d 你的域名.com

# 按提示操作，选择 redirect HTTP to HTTPS
```

### 2. 自动续期
```bash
# 测试自动续期
certbot renew --dry-run

# 添加定时任务（已自动配置）
systemctl enable certbot.timer
```

## 六、防火墙设置

```bash
# 允许 HTTP/HTTPS

# 启用防火墙

ufw status
```

## 九、常见问题

### 1. 刷新页面 404
确保 Nginx 配置中有：
```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

### 2. 静态资源加载失败
检查文件权限：
```bash
chown -R www-data:www-data /var/www/html
chmod -R 755 /var/www/html
```

### 3. 后台无法登录
清除浏览器缓存，或使用无痕模式测试。

## 十、后续更新

当有新版本时，重复步骤三即可：
```bash
# 上传新文件
scp delta-rent-v2.zip root@你的服务器IP:/root/

# 登录并更新
ssh root@你的服务器IP
cd /root
rm -rf /var/www/html/*
unzip delta-rent-v2.zip -d /tmp/
cp -r /tmp/delta-rent/dist/* /var/www/html/
chown -R www-data:www-data /var/www/html
systemctl restart nginx
```
